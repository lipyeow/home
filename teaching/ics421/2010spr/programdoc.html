ICS 421 - Spring 2010 - Programming Assignments Documentation

Updated Mar 12

SQL Data Types Supported

  • SMALLINT
  • INTEGER or INT
  • BIGINT
  • FLOAT(integer), REAL, DOUBLE, DOUBLE PRECISION, FLOAT
  • DECIMAL(prec,scale) OR DEC(prec,scale) or DECIMAL or DEC, NUMERIC, NUM
  • CHARACTER(integer) or CHAR(integer) or CHARACTER or CHAR
  • VARCHAR(integer), or CHARACTER VARYING(integer), or CHAR VARYING(integer)
  • DATE
  • TIME
  • TIMESTAMP

Fragment of SQL Supported

SELECT projection-list
FROM table-list
WHERE condition-list
  • projection-list can be *, a list of column-names, or list of column-names and column function calls. Column-names of which can be qualified with a table alias (aka range variable). Examples of column functions are aggregation functions:sum(amount) etc
  • table-list is a list of tables, optionally with aliases specified using "AS alias"
  • condition-list is a list of predicates of the form
    • columnname operator constant
    • columnname operator function(constant)
    • columnname operator columnname
    columnnames may be qualified with table aliases
