  • Other Reference Books:
    • Principles of Distributed Database Systems (2nd Ed.). M. Tamer Ozsu and Patrick Valduriez. Prentice Hall. ISBN: 0-13-659707-6
    • Data Mining (2nd Ed.). Jiawei Han and Micheline Kamber. Morgan Kaufmann. ISBN: 1-55860-901-6
  • Objective: Students meeting the pre-requisites should already have a basic understanding of databases software and a working knowledge of using database technology for simple applications. At the end of this course the successfull student should
    • have gained a deeper understanding of database technologies including more advanced topics such as performance tuning, data warehousing, parallel and distributed databases.
    • be able to install and deploy DBMS software
    • be able to program using database APIs
  • Late policy: work submitted past due date and time will receive zero credits.
  • Pre-requisites: ICS 321 ICS 311
  • A (portable) computer is required for this course
